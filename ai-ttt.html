AI Tic-Tac-Toe

A neural-network-themed tic-tac-toe game built with ClojureScript and Reagent.

About this project

An enhanced version of the classic Tic-Tac-Toe game featuring an animated neural network background, multiple difficulty levels, and a sleek dark UI. Built with ClojureScript, Reagent, and HTML5 Canvas.
